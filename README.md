# ZenNIO-Example

Example REST API implementation with SQLite.


## Dependencies

### macOS
```
brew install nghttp2
```

### Ubuntu 18.04
```
apt-get install -y git curl libatomic1 libicu60 libxml2 libz-dev pkg-config clang netcat-openbsd lsof perl nghttp2 libnghttp2-dev
```

### Run
```
swift run

```

### Open
```
http://localhost:8080

```
